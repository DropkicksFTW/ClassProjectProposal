Class Project Proposal

  1. Product Name: Button of doom
  2. Team Members: Michael Santay
  3. Application Overview: This is a game with for 4 players, there are 5 buttons and each player takes pressing buttons one of which eliminates the player that presses its. Every player that is elimineted also removes a safe button.
  4. Technology Overview: Arduino Uno, C++ Programming, LEDs, Buttons.
  5. Proposal Last Updated: 04/17/2026
  6. Feature List:
    • Random sequence generation
    • Button input detection
    • LED and sound feedback
  7. Future Feature List: Colored buttons with built in LEDs.
  8. Basic Technical Features: Using arrays to store the pattern and "for" loops to play the pattern back and check the user's input.
  9. Advanced Technical Features: Using random variables for each button to make and creating an LED system to indicate if a button hit was safe or bad.
