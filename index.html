Class Project Proposal

  1. Product Name: Arduino Sequence Master
  2. Team Members: Michael Santay
  3. Application Overview: This is a hardware game where the Arduino creates a random sequence of lights and sounds. The player has to use buttons to repeat the sequence perfectly to move to the next level.
  4. Technology Overview: Arduino Uno, C++ Programming, LEDs, Buttons, and Git for version control.
  5. Proposal Last Updated: 03/24/2026
  6. Feature List:
    • Random sequence generation
    • Button input detection
    • LED and sound feedback
  7. Future Feature List: Add a high-score screen and a "two-player" mode where players challenge each other.
  8. Basic Technical Features: Using arrays to store the pattern and "for" loops to play the pattern back and check the user's input.
  9. Advanced Technical Features: Using a state machine to handle the game flow and custom functions for the buzzer sounds.
