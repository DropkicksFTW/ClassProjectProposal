Class Project Proposal

  1. Product Name: Arduino Sequence Master: Logic & Memory Challenge
  2. Team Members: Chris, Michael Vincent Santay
  3. Application Overview: This project is a hardware-based puzzle game that generates randomized numeric or color-coded sequences. The player must use buttons to input the correct sequence within a set time limit. The core of the game focuses on sequence selection and algorithmic loops to manage game states and verify user accuracy.
  4. Technology Overview:
    • Hardware: Arduino Uno, Input Buttons, LED Output Array, Piezo Buzzer.
    • Software: C++ (Arduino Sketch), Git for version control.

Proposal Last Updated

March 24, 2026

Feature List

  • Dynamic Sequence Generation: Uses C++ random() logic to create a new challenge every round.
  • Multi-Level Difficulty: Incremental loops that increase sequence length as the player progresses.
  • Real-time Feedback: Visual (LED) and Auditory (Buzzer) cues for correct/incorrect inputs.
  • Score Tracking: Keeps track of the highest level reached during the session.

Future Feature List

  • Save State: Using EEPROM to save high scores even after the Arduino is powered off.
  • Custom Mode: Allowing a second player to "program" a sequence for the first player to solve.
  • LCD Integration: Moving from LED blinks to a full text-based menu system.

Basic Technical Features

  • Sequence Selection Logic: Utilizing C++ Arrays to store and compare "Target" vs "Player" data.
  • Iterative Loops: for and while loops to cycle through sequences and check for input timeouts.
  • Conditional Branching: if/else structures to determine win/loss conditions based on sequence matching.

Advanced Technical Features

  • State Machine Architecture: Managing game flow (Idle -> Playback -> User Input -> Result) using switch statements.
  • Debouncing Logic: Software-level timing to prevent "double-counting" a single physical button press.
